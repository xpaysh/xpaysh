🎯 What we ship

🤖 Multi-protocol from day one

Every plugin speaks all three open agentic-commerce specs out of the box

  • ACP — OpenAI · Stripe · Meta (TSC)
  • UCP — vendor-neutral, RFC 9421 signed requests, MCP/A2A bindings
  • AP2 — Google, mandate-based
  • No protocol lock-in for the merchant

💳 Rail-agnostic

Settle via the payment rail your buyers already use

  • Cards (Stripe, Adyen, Braintree, …)
  • Stripe MPP (Machine Payments Protocol)
  • x402 (HTTP 402 stablecoin rail)
  • Direct stablecoins (Bridge, Tempo, Privy)

🔍 Real-standard discovery only

No invented well-known URIs; CI-linted against fictitious standards

  • /llms.txt (llmstxt.org)
  • schema.org JSON-LD on PDPs & listings
  • robots.txt allowlist for real AI crawlers
  • /.well-known/agent-card.json (A2A, watchlist)

🛠️ Active projects

🛒 agentic-commerce-for-woocommerce

Live (v0.1.7+, GPLv2). The reference implementation for the family.

  • ACP + UCP + AP2 discovery and endpoints
  • Cart deeplink → existing WC checkout
  • Non-custodial — your WC payment gateway settles
  • Free until first AI-attributable sale

🧩 agentic-commerce-for-commercetools

Phase B-1 — headless / Connect Service for composable storefronts

  • TypeScript Node service alongside the storefront
  • Backed by commercetools Cart and Order resources
  • Strongest agentic search-demand signal of Phase B
  • BigCommerce (B-2), Magento (B-3), Shopify-app (B-4), SFCC (B-5) follow
